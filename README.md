## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/ERockzilla/medev_ai_25.git
cd medev_ai_25

# Install dependencies
npm install

# Run development server

```

### Build for Production

